Listas vs Pilhas

Lista (Array):

  • Acesso Aleatório: Pode acessar qualquer elemento pelo índice
  • Tamanho Dinâmico: Pode crescer ou diminuir
  • Inserção/Remoção: Em qualquer posição (custo O(n) no pior caso)
  • Operações Comuns: Get, Set, Insert, Remove, Search

Pilha (Stack):

  • LIFO: Last In, First Out (Último a entrar, primeiro a sair)
  • Acesso Restrito: Só pode acessar o topo
  • Operações Básicas: Push (empilhar), Pop (desempilhar), Peek (ver topo)
  • Aplicações: Undo/Redo, Chamadas de função, Navegação

Propriedades

  • Complexidade Temporal:
  • Lista: Acesso O(1), Inserção/Remoção O(n) no pior caso
  • Pilha: Push O(1), Pop O(1), Peek O(1)
  • Complexidade Espacial: O(n) para ambas
  • Uso de Memória: Contígua para ambas
