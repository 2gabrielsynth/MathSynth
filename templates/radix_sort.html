Como o Radix Sort Funciona

O Radix Sort é um algoritmo de ordenação não-comparativo que ordena números processando dígitos individualmente.

Passos (LSD - Least Significant Digit):

  1. Encontrar máximo: Determinar o número máximo para saber quantos dígitos processar
  2. Ordenar por dígitos: Para cada dígito (do menos ao mais significativo):
    • Distribuir números em buckets (0-9 para base 10)
    • Coletar números dos buckets na ordem
  3. Repetir: Repetir para cada posição de dígito

Complexidade Temporal:

  • Melhor caso: O(nk) onde k é número de dígitos
  • Caso médio: O(nk)
  • Pior caso: O(nk)

Complexidade Espacial: O(n + k)

Estável: Sim

In-place: Não

Propriedades do Radix Sort

  • Algoritmo Estável: Mantém a ordem relativa de elementos com mesmo valor
  • Base Dependente: Funciona com qualquer sistema numérico (binário, decimal, etc.)
  • LSD vs MSD: Pode ser implementado do dígito menos (LSD) ou mais (MSD) significativo
  • Sub-rotina: Usa Counting Sort ou Bucket Sort para cada dígito
