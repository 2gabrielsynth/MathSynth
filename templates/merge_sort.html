Como o Merge Sort Funciona

O Merge Sort é um algoritmo de ordenação que usa a técnica "Dividir para Conquistar".

Passos:

  1. Dividir: Divide o array pela metade recursivamente
  2. Conquistar: Ordena cada metade recursivamente
  3. Combinar: Junta as metades ordenadas em um único array ordenado

Complexidade Temporal:

  • Melhor caso: O(n log n)
  • Caso médio: O(n log n)
  • Pior caso: O(n log n)

Complexidade Espacial: O(n)

Estável: Sim
