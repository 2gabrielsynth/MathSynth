Como o Heap Sort Funciona

O Heap Sort é um algoritmo de ordenação baseado na estrutura de dados Heap (Árvore Binária).

Passos:

  1. Construir um Heap: Transformar o array em um heap (max-heap para ordenação crescente)
  2. Extrair máximo: Trocar o elemento raiz (maior) com o último elemento
  3. Re-heapify: Reorganizar o heap para manter a propriedade heap
  4. Repetir: Repetir os passos 2-3 até que o heap tenha tamanho 1

Complexidade Temporal:

  • Construir heap: O(n)
  • Ordenar: O(n log n)
  • Total: O(n log n)

Complexidade Espacial: O(1) (in-place)

Estável: Não

In-place: Sim

Propriedades do Heap

  • Árvore Binária Completa: Todos os níveis estão completamente preenchidos
  • Propriedade Heap: Para max-heap: pai ≥ filhos; para min-heap: pai ≤ filhos
  • Representação como Array: Para índice i:
    • Pai: ⌊(i-1)/2⌋
    • Filho esquerdo: 2i + 1
    • Filho direito: 2i + 2
